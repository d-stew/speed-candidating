About Speed CandiDating

Speed CandiDating was built using HTML5 and CSS3, JavaScript and jQuery, along with Twitter's Bootstrap. Candidate profiles were assembled using information from sites like InsideGov, ProCon.org, and OnTheIssues. Each candidate's policy profile is hard-coded and compared with user responses using JavaScript. The site also features Facebook's API to give users the ability to share Speed CandiDating with friends.
